✦ Completed
Standard Revit Title Block — Recreated with Shared Parameters Matching AutoCAD Template
The client's team had been working with a standard AutoCAD title block for all their drawings — containing a defined set of fields such as project name, drawing number, revision, date, drawn by, checked by, and other project-specific data. When transitioning work into Revit, there was no equivalent title block set up, meaning drawing sheets lacked standardised information and couldn't be linked to the Revit project data. I created a fully standards-compliant Revit title block family from scratch, then set up shared parameters covering every field from their existing AutoCAD title block — ensuring a like-for-like match. This meant all Revit sheets could now carry the same structured drawing metadata, enabling consistent documentation across both platforms.
⚠ Before
All drawings produced using an AutoCAD title block template. When working in Revit, no matching title block existed — sheets had no standardised fields, no shared parameters, and drawing data couldn't be linked or scheduled within the Revit environment.
✓ After
A custom Revit title block family created from scratch with shared parameters mapped exactly to the client's AutoCAD title block fields — consistent drawing metadata, fully linked to Revit project data, enabling standardised sheet output across both AutoCAD and Revit workflows.

✦ Major Implementation
Lighting Layout — Revit-Native Placement, Level-Separated Sheets, Auto Elevation Annotation & Circuiting
This was the most significant workflow transformation in the project. Previously, the team placed light fixture blocks directly in AutoCAD, manually typed circuit and approximate level attributes against each fixture, and created sheets in AutoCAD — a fully manual, error-prone process with no intelligence. I redesigned the entire workflow: light fixtures were placed in Revit as parametric families, levels were driven automatically by Revit's elevation data, and all sheets were produced inside Revit. When annotation collisions occurred due to multi-level fixtures appearing on a single sheet, I proposed and implemented level-separated sheets — one sheet per floor level — eliminating annotation overlap entirely. Circuiting was done directly in Revit using the custom panel family I built in Implementation 01. I also proposed a lighting schedule to be embedded in the sheet, and suggested power circuiting as the next step for upcoming projects.
